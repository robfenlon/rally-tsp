# SaddleSore SS1600K — IBA Certification Checklist

> Rides fail certification on **paperwork**, not riding. The IBA verifies your claim
> from receipts and a witnessed log. Get the documentation right or the ride doesn't count.
> Always confirm current rules at ironbutt.com / ironbutt.co.uk before you ride — this
> checklist reflects the long-standing requirements but the IBA is the source of truth.

## Before the ride

- [ ] Read the **current** SS1600K rules from the IBA (UK/Ireland chapter if applicable).
- [ ] Download/print the official **ride log sheet** and **witness form**.
- [ ] Line up a **start witness** and an **end witness** (can be different people).
      They sign the form with name, signature, and contact details.
- [ ] Confirm your route gives **≥ 1,600 km start-receipt-to-end-receipt** with buffer.
- [ ] Register bike for **M50 eFlow toll** (avoids penalty hassle).
- [ ] Phone fully charged + power bank; GPS track app ready (see below).
- [ ] Know which stations on your corridors are **24h** and print/save pump receipts policy.

## The start (gets you on the clock)

- [ ] Get a **computer-printed fuel/purchase receipt** at the start showing
      **date, time, location, and ideally the station name**. This timestamp **starts the clock**.
- [ ] If the receipt lacks a location, also buy something that prints the address,
      or photograph the station sign with the bike + odometer.
- [ ] Have the **start witness sign** the log with the start odometer reading and time.
- [ ] Photograph: odometer at start, the start receipt, the station.

## During the ride (build the evidence chain)

- [ ] Keep **every** fuel receipt, in order. Don't bin any of them.
- [ ] At each **turn / corner point** (Newry, Cork, Galway, Waterford, Limerick), get a
      timestamped receipt — these prove you actually rode the distance, not a short loop.
- [ ] Fill in the **log sheet** at each stop: location, odometer, time. Do it every stop,
      not from memory later.
- [ ] Run a **GPS tracking app** (e.g. a continuous track export / spot tracker) as
      corroboration. Not always mandatory but settles disputes instantly.
- [ ] If a receipt prints with **no time** (some pumps), buy inside to get a timed till receipt.

## The finish (stops the clock)

- [ ] Final **computer-printed receipt** at the finish — date, time, location.
      Elapsed = finish timestamp − start timestamp must be **< 24h 00m**.
- [ ] **End witness signs** the log with end odometer and time.
- [ ] Photograph: final odometer, final receipt, the station.
- [ ] Sanity check on the spot: **end odo − start odo ≥ 1,600 km** AND elapsed < 24h.

## Submission pack

- [ ] Completed, signed **log sheet** (start + end witness sections).
- [ ] **All receipts** in chronological order (scan/photograph as a single ordered PDF).
- [ ] **GPS track** export if used.
- [ ] Photos (odometer start/end, key receipts).
- [ ] Submit per current IBA process and pay any certification fee.

## The two numbers that must both be true

| Check | Requirement |
|-------|-------------|
| Distance | end receipt location − start receipt location ≥ **1,600 km** |
| Time | finish timestamp − start timestamp < **24h 00m** |

Everything in the route plan and pace model exists to make **both** of these
comfortably true with margin to spare.
